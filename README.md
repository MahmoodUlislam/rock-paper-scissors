# React Calculator

A calculator project built with React.

![Alt img](https://i.imgur.com/vhtRGoQ.png)

## Installation

1. `git clone https://github.com/madzadev/react-rock-paper-scissors.git`

2. `cd react-rock-paper-scissors`

3. `npm install`

4. `npm start`
